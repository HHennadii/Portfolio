005 : Steampunk Gun

Blender 3D / UVlayout / Marmoset Toolbag / Substance Painter / Photoshop

In this case, I would like to highlight one of my first works in the field of game development. This project was aimed at creating weapons in the steampunk style. During this project, a high-poly model was created that had more than 10m polygons, in the process of optimization and baking, the size of the model was reduced to 40k polygons while maintaining the original quality. This project was the start of the collaboration with DS, which is described in the previous case.
